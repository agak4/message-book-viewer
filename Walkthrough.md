# Walkthrough

- 사이드 컨트롤 패널이 페이지 첫 로드 시 처음 6초 동안 화면에 나타나도록 `.show-initial` 초기화 클래스를 새로 추가했습니다.
- `script.js`에서 컨트롤 패널 컴포넌트를 생성할 때 함께 요소에 할당하고, 자바스크립트의 `setTimeout` 기능을 활용하여 6000ms 후에 클래스를 삭제하여 패널이 다시 화면 밖으로 깔끔하게 들어갈 수 있도록 조치하였습니다.
- 호버링 상태(`:hover`)와 함께 `.show-initial` 상태도 동일하게 사이드 화면 우측(`right: 0;`)에 고정될 수 있도록 CSS 속성을 함께 그룹화하여 유지보수에 용이하도록 결합했습니다.
